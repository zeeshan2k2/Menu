# Menu  

**Menu** is an iOS application built with **SwiftUI**, focused on building a clean, structured food ordering interface with multiple screens and state-driven UI updates.  

---

## Features  

- Menu listing with categorized items  
- Detailed view for individual menu items  
- Account screen UI  
- Order screen flow  
- Multi-screen navigation using SwiftUI  
- State-driven UI updates

## Tech Stack  

- **Language:** Swift  
- **Framework:** SwiftUI  
- **Architecture:** MVVM  
- **State Management:** ObservableObject, @Published  
- **Navigation:** NavigationStack
